# Calculator App Using Flutter

A simple calculator app built using Flutter, a cross-platform mobile development framework. This app allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
 - Addition
 - Subtraction
 - Multiplication
 - Division

## Getting Started
To run this application on your local machine, make sure you have Flutter installed. If you don't have Flutter installed, you can follow the instructions on the official Flutter website: https://flutter.dev/docs/get-started/install

## Prerequisites
 - Flutter SDK
 - Android Studio / Xcode (for iOS development)
 - Clone the repository
 - bash
 - Copy code
 - git clone https://github.com/shreeramkedlaya/calculator.git
 - cd calculator_app
 - Run the app
 - bash
 - Copy code
 - flutter run


## How to Use
1. Launch the app on your device or emulator. You will see a simple  calculator interface with numeric buttons and arithmetic operation buttons.

2. Tap the numeric buttons to input numbers.
3. Tap the arithmetic operation buttons to perform calculations.
4. The result will be displayed at the top of the screen.
